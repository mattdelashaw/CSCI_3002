Cloud Computing

The Cloud. The omnipotent, non-existent... thing we rely on, for just about everything. One of the biggest advantages of cloud computing is space. For example, instead of filling your house with all the hardware and cooling equipment you would need to run some big project your working on, you can do the same with a single interface, most likely, and the cloud. Cloud computing relies on the idea of virtualization. Virtualization takes advantage of idle hardware components and utilizes them to aid in additional tasks. You can do the same thing at home with your own computer. If you wanted, you could take a desktop computer running windows that youre playing a video game on, and at the same time boot up a Linux operating system on the same machine to write that big paper that's due tomorrow for that Ethical House Fly Breeding class you're taking. This example could slow your gaming performance and wouldn't be advisable, but neither is taking Ethical House Fly Breeding. It's the same concept though. Spliting the hardware resources of a machine to perform additional tasks, on an independent operating system.
